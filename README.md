# Emailable Tailwind Core Styles

TailwindCSS theme and core styles for Emailable.

## Style Guide

### Colors

| UI component                              | Light mode         | Dark mode                                               |
| ----------------------------------------- | ------------------ | ------------------------------------------------------- |
| Background - General                      | `White`            | `Slate 950`                                             |
| Background - General alternative          | `Slate 50`         | `Slate 900`                                             |
| Background - Popovers                     | `White`            | `Slate 800`                                             |
| Buttons & links                           | `Primary 500`      | `Primary 500`                                           |
| Borders                                   | `Slate 100`        | `Slate 800`                                             |
| Headings & bold text                      | `Slate 800`        | `White`                                                 |
| Subheadings & body text                   | `Slate 600`        | `Slate-300` / `Primary 300` (Gradient backgrounds only) |
| Article body text                         | `Slate 700`        | `Slate-300`                                             |

### Gradients

| Name              | Style                          |
| ----------------- | ------------------------------ |
| Dark              | `Primary 800` to `Primary 950` |
| Darker            | `Primary 900` to `Primary 950` |
| Purple            | `Primary 500` to `Primary 800` |

## Minimization

It's important that we minimize additions to the core styles to prevent
unnecessary bloat. We should carefully review all additions to ensure we keep
this slim.
